# 🤖 Freelance Bot — Автоматический поиск заказов

Бот-парсер для автоматического поиска фриланс-заказов в Telegram и VKontakte.

## Что умеет
- Парсит фриланс-каналы Telegram и группы VK
- Разделяет заявки на: 🟢 Горячие (ищут исполнителя) / 🔵 Холодные (потенциальные клиенты) / 🔴 Спам
- Отправляет уведомления с готовым текстом отклика прямо в Telegram
- Запускается автоматически в 8:00 каждый день
- Выгружает результаты в Excel в конце дня

## Технологии
- Python 3.12
- requests + BeautifulSoup — парсинг
- APScheduler — расписание
- Telegram Bot API — уведомления
- VK API — парсинг ВКонтакте
- tkinter — мини-окно управления
- openpyxl — выгрузка в Excel

## Установка
1. Клонируй репозиторий
2. Скопируй `.env.example` в `.env` и заполни своими ключами
3. Установи зависимости: `pip install -r requirements.txt`
4. Запусти: `python main.py`
